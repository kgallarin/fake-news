# For Manatal

Simple news gallery

Created with Vue.js

## Project setup
```
npm install or npm i
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

## Built With

* [Vue](https://vuejs.org/) 
* [Vuetify](https://vuetifyjs.com/en/) 
* [Masonry Layout]

* [News API](https://newsapi.org/)
